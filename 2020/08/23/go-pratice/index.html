Go 的第一支程式

使用 VSCode 去練習 Go 語言

Go 的環境
要先去 Go 官網下載安裝包
並且設定環境變數
然後安裝相依套件

寫完程式後
在 VSCode 上的終端機上
要先把剛打好的程式編譯

輸入完
go build hello.go
會新增一個編譯完的 exe 檔

然後在輸入
.\hello.exe
就可以看到執行結果

以上是 Go 語言的 Hello World 練習
