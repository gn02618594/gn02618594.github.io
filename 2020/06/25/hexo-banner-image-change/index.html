hexo更換頂部圖片

當你主題已經設定好之後，但是頂部的圖片卻不喜歡
想要換一個自己的寵物、喜歡的風景、愛吃的食物、或是想要幫自己家裡打個廣告
那該怎麼做呢

本篇文章是以 Hexo 預設的主題去說明的，但其他主題修改的地方可能不一樣

首先進到自己的 hexo 資料夾

然後移動到
themes/主題/sources/css

到這裡之後 可以看到有個images的資料夾
把想要換的圖放進去

暫時命名叫 banner.jpg

已經有圖了，接下來呢

css 資料夾下有個 _partial 的資料夾
有個檔案叫做 header.styl

把它打開來編輯，並找到#banner

這時候我們可以看到

background: url(/css/images/banner.jpg)

把 banner.jpg 改成你圖檔的名稱就可以了
