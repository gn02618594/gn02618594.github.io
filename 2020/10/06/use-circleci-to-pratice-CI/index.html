使用 CircleCI 練習 CI/CD

上次練習了測試
這次想練習看看 CI 怎麼使用

一開始先進去 circleci 的官網
https://circleci.com/
選擇登入
https://circleci.com/vcs-authorize/

然後會要你使用 github 或是 bitbucket 連動
我這邊是選擇 github

然後要求授權

選擇帳戶

然後就可以看到你目前 github 全部的專案

你要選一個去 Set Up Project

他就會要求你建立一個 yml 檔
yml 檔就是你 CI 要使用的腳本
接著要選你使用的程式語言

加入之後你上傳的當下
你每次這個專案更新上 github 之後
他就會偵測並且執行寫在腳本的東西


本來還要加再腳本加上

上傳後會傳訊息到 slack
