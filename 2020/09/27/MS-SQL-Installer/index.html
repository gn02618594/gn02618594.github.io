SQL-SERVER 安裝

因為想要使用 LocalDB
但是自己電腦怎麼嘗試都沒辦法
也找很多文章也有詢問別人
還是找不出原因
所以就想說重新安裝看看有能不能使用
也順便記錄一下

無法使用 LocalDB 的錯誤訊息
))

本來是使用 2014 版本
想趁著這次機會裝更新版的
但發現….
2016、2017、2019 都不支援 WIN7
所以還是只能裝 2014

執行安裝檔執行新增 SQL SERVER

選擇要安裝的功能

設定伺服器名稱
有參考文章發現
不知為什麼這邊的安裝的執行個體是空的
會不會沒辦法用 LocalDB 是跟這邊有關


設定密碼
也可以選擇 WINDOWS 驗證
但覺得如果是要給程式連接資料庫使用
有設置密碼可能沒那麼多問題
不知 WINDOWS 驗證是透過什麼去驗證的

這樣就安裝完了

組態管理員也有執行中

結果又是不一樣的錯誤訊息

看來還要在試一下了
