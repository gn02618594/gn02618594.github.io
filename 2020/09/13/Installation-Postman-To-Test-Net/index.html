安裝 POSTMAN

在學 WEB API 的路上
發現很多大神寫的文章好像都有使用 POSTMAN 這個工具來測試
所以~想要來安裝並且試試看能不能了解功能如何使用

使用 POSTMAN 可以透過
桌面應用程式
以及
Chrome 的線上應用程式

首先先進官網

然後我開始找
都沒看到桌面應用程式要去哪裡下載

所以!!
我就先 Sign in 了!!

Sign in 後設定一下名稱跟職位以及使用 POSTMAN 的用途之後
就可以成功開始使用 POSTMAN 了

後來發現如果要下在桌面應用程式的話
要在官網的頁面下

https://www.postman.com/

拉到最底部
在這邊就可以看到 Download the app
點進去就可以下載
POSTMAN 的桌面應用程式了

所以使用 POSTMAN 可以有三種方式

1.桌面應用程式
2.線上應用程式
3.網頁上直接使用

使用 httpbin.org/get 來測試(這是網頁版的 PSOTMAN)
