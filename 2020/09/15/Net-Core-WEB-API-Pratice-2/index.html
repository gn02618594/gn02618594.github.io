使用 .NET Core 練習 ASP.NET WEB API 2

接續上一次

要裝兩個東西
我是使用 管理方案的 NuGet 套件去安裝的

一個是
Microsoft.EntityFrameworkCore.SqlServer
這是為了讓專案便利使用 SqlServer 而裝的

一個是
Microsoft.EntityFrameworkCore.SqlServer
這是為了可以在 IDE 上直接使用 scaffolding、migration 命令
scaffolding 很像是為了 快速產生 Controller 內容與檢視頁
migration 是為了移轉，但移轉的定義看了官方文件還是有點模糊
https://docs.microsoft.com/zh-tw/aspnet/core/data/ef-mvc/migrations?view=aspnetcore-3.1
https://docs.microsoft.com/zh-tw/ef/core/managing-schemas/migrations/?tabs=dotnet-core-cli
可能之後使用到了才會比較了解

那兩個都裝完後
可以在方案總管中的 相依性-套件中看到
